#  CULT Sport Accessories – Product Label Generator

This tool allows you to automatically generate **50×60 mm barcode/MRP labels** for CULT Sport Accessories using data from an Excel file.

The form auto-fills from Excel, previews the label, and lets you download or print **high-quality text-based PDFs**.

---

##  Features

- Upload Excel file to auto-fill product details  
- Live label preview  
- Print single label (**50×60 mm**)  
- Download all labels (one per page)  
- Navigate between multiple rows in Excel  
- Barcode automatically rendered  
- Supports required Excel template  

---

## How to Use

1. **Prepare your Excel file** with the exact column names.  
2. **Upload the Excel file** in the tool.  
3. The form will **auto-fill** with the first row.  
4. Use **Previous / Next** buttons to navigate product rows.  
5. Click **Update Label** to refresh the live preview.  
6. Click **Print Current Label** to print a **single 50×60 mm label**.  
7. Click **Download All Labels as PDF** to export all labels as one PDF (one label per page).
