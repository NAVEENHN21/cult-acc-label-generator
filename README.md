CULT Sport Accessories – Product Label Generator

This tool allows you to automatically generate 50×60 mm barcode/MRP labels for CULT Sport Accessories using data from an Excel file.
The form auto-fills from Excel, previews the label, and lets you download or print high-quality text-based PDFs.

Features:
•	Upload Excel file to auto-fill product details
•	Live label preview
•	Generate high-quality text-based PDF labels (not images)
•	Print single label (50×60 mm)
•	Download all labels (one per page)
•	Navigate between multiple rows in Excel
•	Barcode automatically rendered
•	Required Excel Format

Prepare your Excel file with the following exact column headers:
Month and Year of Manufacture
Brand
Article Number
Article Name
Colour
Net Quantity
Size
Dimension
Country of Origin
Gender
MRP
Barcode Number

Each row will generate one label.
How to Use
•	Prepare your Excel file with the exact column names listed above.
•	Upload the Excel file in the tool.

•	The form will auto-fill with the first row.
•	Use Previous / Next buttons to move between product rows.
•	Click Update Label to refresh the live preview.
•	Click Print Current Label to print a single 50×60 mm label.
•	Click Download All Labels as PDF to export all labels as one PDF (one label per page).
